YouTube to Facebook

1. What is YouTube to Facebook?

YouTube to Facebook is a web app, which can post YouTube video directly to Facebook as a video, not link.

2. How to do it?

Firstly, you will need a Facebook access token, which will allow us to publish post as you. And a YouTube video link, which is the link of YouTube video you want to publish to Facebook.

3. How to get Facebook access token?

In order to get your Facebook access token, go to https://developers.facebook.com/tools/explorer/ and log in if you are not. And if you are logged in, you will see Facebook Graph API Explorer. Now click "Get Token" and then "Get User Access Token". And in the "Select Permissions" box, tick the check box next to "publish_actions", and click " Get Access Token". Click "Continue" and then "OK" when you are asked to allow. After this, you will be redirected back to Graph API Explorer page and now you see your access token is in the "Access Token" box. You can copy by click on it. You can also noticed "i" mark in the "Access Token" box, this is the information about this access token. You can check this by clicking on it, where you can see whether this access token is still valid or not, token's Expiration Time, and allowed permissions associated with this token.

4. How to get YouTube video url?

You can get YouTube video url in the address bar of browser while you are watching this video. Or if you are on YouTube app, just click "Share" button, and then "Copy link".

5. I have Facebook access token and YouTube video url. How to start publish YouTube video to Facebook?

Go to http://video.formyanmar.tk , fill in the form with your access token and YouTube video url and click "Submit". Done!
